# E-Commerce-Sales-Performance-Dashboard
- Developed an interactive dashboard to analyze YTD(Year to Date Sales), Profit, Quantity, and YoY(Year on Year) Sales performance and built DAX measures and transformed data using Power Query for accurate KPI reporting. 
- Designed interactive visualisations for sales by category, region, state, shipping mode, and top/bottom-performing
products.
– Implemented dynamic slicers and drill-down features to enable data-driven business insights.
